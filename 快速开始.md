# 快速开始

## 方法一：直接使用（推荐）

1. 直接双击打开 `index.html` 文件
2. 或者在浏览器中打开该文件
3. 开始使用所有功能

就这么简单！无需安装任何依赖或启动服务器。

## 方法二：使用本地服务器（可选）

如果你想通过 HTTP 协议访问（某些浏览器的高级功能可能需要），可以：

### 使用 Python（如果已安装）
```bash
# Python 3
cd d:\workspace\5images\case5-images01
python -m http.server 8000

# 然后在浏览器访问: http://localhost:8000
```

### 使用 Node.js（如果已安装）
```bash
# 安装 http-server（仅需一次）
npm install -g http-server

# 启动服务器
cd d:\workspace\5images\case5-images01
http-server

# 然后在浏览器访问: http://localhost:8080
```

### 使用 VS Code Live Server
1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"
